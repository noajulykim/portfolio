# AI·데이터 교육사업 운영 포트폴리오

공공기관·기업의 AI·데이터·AX 교육사업 운영 기록 (2022–2026).

## 폴더 구조 (이대로 업로드하세요)

```
(저장소 루트)
├── index.html      ← 메인 페이지
├── README.md       ← 이 파일 (선택)
└── images/         ← 사진 폴더 (반드시 함께 업로드)
    ├── env-1.jpg
    ├── env-2.jpg
    ├── env-3.jpg
    ├── env-4.jpg
    ├── env-5.jpg
    └── env-6.jpg
```

> `index.html`은 `images/` 폴더 안의 사진을 불러옵니다. 두 가지를 **함께** 올려야 사진이 보입니다.

## 배포 방법 (GitHub Pages)

1. 저장소 **Code** 화면에서 **Add file → Upload files** 클릭.
2. 압축을 푼 폴더 안의 `index.html`, `README.md`, 그리고 **`images` 폴더를 통째로** 드래그&드롭. (폴더를 드래그하면 구조가 그대로 올라갑니다)
3. 아래 **Commit changes** 클릭.
4. **Settings → Pages → Source: Deploy from a branch → Branch: main / (root) → Save.**
5. 1~2분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 공개됩니다.

## 메모

- 사진은 가로 한 줄로 정렬되며, 높이를 통일하고 가로폭은 자동이라 비율이 유지됩니다. 폭이 넘치면 가로 스크롤됩니다.
- 외부 의존성은 Google Fonts(IBM Plex)뿐입니다.
